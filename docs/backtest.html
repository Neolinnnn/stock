📜 策略規則

進場條件

第一層：族群掃描雙篩選通過（qualified）
— 族群 20 日回報排名前段 + RSI5 > 60
— 個股 RSI5 40-65 + 20日漲幅 >3% + CV夏普 >0.5

第二層：趨勢評分引擎 ≥ 60 分
— MA排列 / 乖離率 / 量能 / MACD / RSI / 支撐
— 搭配族群強弱 (+18/-22) × CV夏普 (+15/+8)

進場時機：訊號日次日開盤買入

出場條件

主要出場：收盤跌破 MA10（10日均線）
→ 次日開盤出場

保護停損：盤中低點跌破 -20%
→ 當日以 -20% 價位出場

最長持倉：60 個交易日強制出場

同股有未出場倉位時不重複進場

🔍 回測觀察
✅ 非對稱報酬結構（核心優勢） 勝率 42.9% 但平均報酬為正（+2.96%）。原因：平均獲利 +13.95% vs 平均虧損 -5.28%，盈虧比達 2.64x。 MA10 出場讓大波段充分奔跑（華邦電 +46-47%、創見 +33%、矽格 +25%），同時小幅虧損被快速截斷。
✅ 最強個股：2344 華邦電 × 2451 創見 華邦電 4勝2敗，平均報酬 +21.5%；創見 4勝0敗，平均報酬 +19.3%。 兩股在族群強勢 + 高 CV 夏普的雙重加持下，趨勢評分穩定在 60-75 分，訊號質量高。
⚠️ 弱點：市場修正期下的連敗（2026/03） 2026 年 2-3 月南亞科（0勝3敗，均虧 -13.1%）、超豐連續多筆虧損，與當時大盤修正直接相關。 MA10 在急跌中反應較遲鈍，建議搭配大盤過濾（加權指數在 MA20 以上才進場）可減少此類損失。
⚠️ 怡利電（2243）頻繁訊號但勝率低（2W 7L, -2.54%） 連續多日出現訊號，表示族群持續維持強勢，但個股趨勢評分不穩定（49~79 分波動大）。 已加入「同股不重複進場」邏輯，但此股在震盪期進出耗費較多次數。
📊 MA10 出場 vs TP/SL 固定出場比較 TP/SL 策略（左頁籤）：勝率 ~60%、平均持倉 ~5天、多數快速觸及停利。 MA10 策略：勝率較低 43%，但勝利時持倉更長（允許大波段 +30~47%），整體期望值更高。 兩種策略各有適用市況：強趨勢行情 MA10 更佳；震盪行情 TP/SL 勝率更穩。

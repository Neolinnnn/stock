## 2. 環境安裝

### 第一次使用

```bash
# 建立虛擬環境（Windows）
python -m venv twstock-env
twstock-env\Scripts\activate

# 安裝套件
pip install twstock finmind pandas numpy matplotlib seaborn plotly jupyter
```

### 下載所有 GitHub Repo

```bash
bash download_repos.sh
```

### 驗證安裝成功

```bash
python test_examples/01_basic_data_fetch.py
```

## 3. 核心工具庫

| Repo | 主要功能 | 難度 |
|------|---------|------|
| **twstock** | 即時/歷史股價、MA/RSI/MACD 等技術指標 | ⭐ 入門 |
| **FinMind** | 三大法人、融資融券、EPS、月營收等完整資料集 | ⭐⭐ 初級 |
| **node-twstock** | Node.js 版本，可串接前端網頁 | ⭐⭐ 初級 |
| **Taiwan-Stock-KG** | 概念股知識圖譜、產業鏈分析 | ⭐⭐⭐ 中級 |
| **Qlib** | 因子研究、XGBoost/LightGBM 機器學習預測 | ⭐⭐⭐ 高級 |
| **FinRL** | 強化學習自動交易（DDPG/PPO/SAC） | ⭐⭐⭐ 高級 |

**推薦組合（由淺入深）：**

- 快速驗證想法：`twstock` + `FinMind`
- 系統化分析：`FinMind` + 自建資料庫
- AI 策略開發：`Qlib` + `FinRL`

## 4. 策略腳本說明

### 4-1. 個股深度分析（最常用）

```bash
python strategy_templates/05_single_stock_deep_dive.py 5292
```

**輸出內容：**
- 技術面：現價、MA5/20/60/120、RSI、52週位階、均線結構（多頭/空頭/盤整）
- 量能：今日量 vs 20日均量、量比判斷
- Walk-Forward CV 策略驗證：3折歷史勝率、夏普比率
- 綜合買點分數（-6 ~ +8），給出：強烈買進 / 偏多 / 中性 / 偏空
- 建議價位：進場區間、停損參考、第一/第二目標

### 4-2. 每日族群掃描

```bash
# 手動執行
python strategy_templates/07_daily_scan.py

# 或直接雙擊
run_daily_scan.bat
```

**掃描族群（共 7 大類）：**

| 族群 | 代表股 |
|------|--------|
| 光通訊 | 4979 華星光、3450 聯鈞、4906 正文 |
| 記憶體 | 2408 南亞科、2337 旺宏、2344 華邦電 |
| AI伺服器 | 2317 鴻海、2382 廣達、3231 緯創 |
| 其他 | （詳見腳本定義） |

**輸出：** `daily_reports/YYYYMMDD/` 下的 `summary.md`、`summary.json`、族群比較圖 `.png`

### 4-3. 週報生成

```bash
python strategy_templates/08_weekly_summary.py
# 或雙擊 run_weekly_summary.bat
```

彙整本週所有日報，產生族群趨勢週變化圖與 Top Picks 排行。

### 4-4. 批次掃股（含 Walk-Forward CV）

```bash
python strategy_templates/03_batch_scan_with_cv.py
```

對清單內所有股票進行歷史策略驗證，適合篩選出具統計意義的候選股。

### 4-5. 光通訊族群掃描

```bash
python strategy_templates/04_optical_comm_scan.py
```

輸出結果儲存於 `optical_comm_scan.csv`。

### 4-6. 基本策略（學習用）

```bash
python strategy_templates/01_moving_average_crossover.py   # MA 均線交叉
python strategy_templates/02_rsi_reversal.py                # RSI 反轉
```

### 4-7. FinMind 基本面分析

```bash
python strategy_templates/06_finmind_fundamentals.py
```

拉取 EPS、月營收、本益比等基本面資料。

## 5. 日常研究流程

### 盤後例行（每天）

1. 雙擊 `run_daily_scan.bat`
2. 查看 `daily_reports/YYYYMMDD/summary.md`
3. 對有興趣的個股執行深度分析：
   ```bash
   python strategy_templates/05_single_stock_deep_dive.py <股票代號>
   ```

### 週末例行（週五收盤後）

1. 雙擊 `run_weekly_summary.bat`
2. 查看週報圖表與 Top Picks
3. 更新 `watchlist_tracking.md`

### 研究新股票

1. 查觀察清單（`C:\Users\Neo\Downloads\股票清單.md`）確認基本資訊
2. 執行深度分析：
   ```bash
   python strategy_templates/05_single_stock_deep_dive.py <代號>
   ```
3. 用下方提示詞請 Claude 補充質化分析

## 6. 針對個股研究的提示詞範本

### 使用方式

1. 複製下方「主要研究範本」
2. 填入 `【填入區】` 的欄位
3. 貼入 Claude 對話，一次取得全面分析 + 進場判斷 + 同族群比較

---

### 主要研究範本（A+B+C 整合版）

```
【研究標的】
股票代號：[XXXX]
公司名稱：[公司名稱]
所屬族群：[族群名稱，例如：光通訊 / AI伺服器 / 記憶體]
目前股價：[XXX] 元

【公司背景】
產業定位：[一句話描述護城河或市場地位]
主要客戶：[客戶名稱]
近期動態：[最新一則重要消息或營收變化]

【財務快照】
EPS（近四季）：[數值] 元
ROE：[數值]%
毛利率：[數值]%
近期營收趨勢：[年增 or 年減 約 X%，原因]
現金股利：[數值] 元

【技術面數據（貼入腳本輸出）】
（執行 python strategy_templates/05_single_stock_deep_dive.py XXXX 後，將輸出貼在這裡）

---

請依序回答以下三個部分：

## Part A｜全面分析
1. 技術面：目前趨勢（多頭/空頭/盤整）、關鍵支撐與壓力位
2. 基本面：目前估值合理嗎？本益比 vs 同業 vs 成長性
3. 催化劑：近 1-3 個月有哪些利多 / 利空事件
4. 風險：最主要的 2-3 個下行風險

## Part B｜進場判斷
1. 目前技術訊號（均線排列、RSI、量能）是否支持進場？
2. 建議進場區間與停損設定
3. 需要等待哪個確認訊號才進場？（例如：站回 MA20、RSI 脫離超賣）

## Part C｜同族群比較
從 [族群名稱] 同族群中，列出 2-3 支競品，比較：
- 技術面強弱排序
- 基本面（EPS 成長、毛利率）優劣
- 如果只能選一支，推薦哪支？理由為何？

---
投資風格：中長線（3-6 個月），不做當沖，可接受最大 -8% 停損
輸出格式：條列式，每點不超過 3 句話
```

### 提示詞的關鍵原則

| 原則 | 說明 |
|------|------|
| 給代號，不只給名稱 | `5292` 比「華懋」更精確、不易混淆 |
| 貼入技術面腳本輸出 | 讓 Claude 基於數據判斷，而非憑空推測 |
| 說明投資週期與停損 | 短線/中線/長線的建議策略完全不同 |
| 指定輸出格式 | 條列式、每點不超過 3 句話，避免冗長回答 |
| 族群名稱要明確 | Part C 比較才能找到正確的競品 |

## 8. 注意事項

- 所有策略輸出僅供參考，交易前請自行判斷
- 任何策略上線前，建議先回測至少 3 個月
- 過去績效不代表未來表現
- 高波動個股（年化波動率 > 60%）需特別控管部位
- 資料來源為公開市場資料，交易前請確認資料正確性
